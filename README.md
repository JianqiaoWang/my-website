# my-website
personal website of Jianqiao Wang

## Local editing

Start the Hugo preview server:

```bash
npm run serve
```

In a second terminal, start the Decap CMS local backend:

```bash
npm run cms
```

Then open:

```text
http://127.0.0.1:1313/admin/
```

The CMS can edit blog posts, the research page, the group page, and the author profile.
